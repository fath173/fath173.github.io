Tentang Saya

Assalamualaikum wr wr. saya Fathor Rahman adalah seorang mahasiswa dari Politeknik Negeri Jember. Asal saya adalah Kabupaten Probolinggo - Jawa Timur.

Saya tertarik dengan dunia IT sejak bersekolah di SMK Negeri 2 Kraksaan, disana saya mengambil Jurusan Rekayasa Perangkat Lunak.

Karena saya sudah mempunyai basic di SMK maka saya melanjutkannya di kuliah saya dengan mengambil jurusan Teknologi Informasi dengan prodi Teknik Informatika.

Saya sangat menyukai bidang ini, bidang ini begitu menarik, kadang bikin mumet, kadang bikin bahagia ketika telah menyelesaikan error dari sebuah perkodingan, hal itu yang membuat saya tambah semangat dan merasa tertantang menaklukkan perkodingan.
