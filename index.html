About Us

Assalamualaikum wr wr. saya Fathor Rahman adalah seorang mahasiswa dari Politeknik Negeri Jember. Asal saya adalah Kabupaten Probolinggo - Jawa Timur. Saya tertarik dengan dunia IT sejak bersekolah di SMK Negeri 2 Kraksaan, disana saya mengambil Jurusan Rekayasa Perangkat Lunak. Karena saya sudah mempunyai basic di SMK maka saya melanjutkannya di kuliah saya dengan mengambil jurusan Teknologi Informasi dengan prodi Teknik Informatika. Saya sangat menyukai bidang ini, bidang ini begitu menarik, kadang bikin mumet, kadang bikin bahagia ketika telah menyelesaikan error dari sebuah perkodingan, hal itu yang membuat saya tambah semangat dan merasa tertantang menaklukkan perkodingan.

Kampus Merdeka memberikan saya peluang untuk mengasah kemampuan saya dalam bidang Informatika. Sehingga saya mengikuti program kampus merdeka dan memilih untuk mempelajari fullstack web bersama dengan mitra NF Computer. Saya sangat bersyukur telah mengikuti pelatihan ini dan akan saya manfaatkan sebaik mungkin agar tidak sia-sia dalam belajar selama 5 bulan kedepan.

Selamat datang di website pertama saya saat mengikuti pelatihan NF Computer. Ini hanyalah website sederhana yang memang untuk melatih atau mengasah kembali dasar-dasar bahasa pemrograman HTML. Semoga ini menjadi awalan yang baik dalam mengikuti pelatihan dan tetap semangat sampai projek akhir dan lulus test akhir. Kurang lebihnya saya mohon maaf, wassalamualaikum wr wb.
